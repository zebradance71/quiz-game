# 脳トレ・IQクイズゲーム | 30代40代50代の大人向け無料論理パズル

30代〜50代の大人が楽しめる本格的な論理パズル・IQクイズゲームです。無料でプレイでき、会員登録も不要です。

## 特徴

### 🎯 充実した問題数
- **30問の厳選クイズ**: 各難易度10問ずつ、合計30問の良問
- **ランダム出題**: 毎回異なる5問が出題されるため、何度でも楽しめます
- **3段階の難易度**: 初級（一般教養・直感）、中級（論理パズル・計算）、上級（超難問・難解な推論）

### 📝 充実した解説
- **詳しい解説**: 各問題に300文字程度の読み応えのある解説付き
- **知的好奇心を満たす**: なぜその答えになるのか、背景知識も含めて詳しく解説
- **学びながら楽しめる**: 単なるクイズではなく、論理的思考力を養う教材としても活用可能

### 💡 インテリジェント・ヒント機能
- 問題開始から15秒経過後、ヒントボタンが出現
- ヒントを利用した場合、その問題の獲得スコアは半分に
- 戦略的な判断が求められます

### ⏱️ 制限時間システム
- 各問60秒の制限時間
- 早く解くほど高得点（時間ボーナス）
- タイムアップでも次の問題に進めます

### 📚 詳しい解説
- 全問題に分かりやすい解説付き
- 納得感のある丁寧な説明で理解を深められます

### 🎉 特別演出
- 全問正解＋高得点（800点以上）で紙吹雪演出
- デジタル「天才認定証」を表示
- canvas-confettiによる豪華なアニメーション

### 🛍️ おすすめ商品リンク
- 解説画面: 脳トレ本、ブドウ糖菓子、疲れ目対策グッズ
- 結果画面: ビールギフト、最新ガジェットなど「自分へのご褒美」
- Amazonアソシエイト対応（ID: zebradance-22）

## ゲームの流れ

1. **トップページ** → 「クイズを始める」をクリック
2. **難易度選択** → 初級・中級・上級から選択
3. **クイズプレイ** → ランダムに選ばれた5問に挑戦
   - 各問60秒の制限時間
   - 15秒経過後ヒント利用可能（スコア半減）
4. **解説確認** → 詳しい解説とおすすめ商品
5. **結果発表** → 全問正解＋高得点で特別演出！

## スコアリングシステム

- **基本点**: 初級100点、中級200点、上級300点
- **時間ボーナス**: 残り時間 × 2点
- **ヒントペナルティ**: ヒント使用時は獲得スコア半分

### 特別演出の条件
- 全5問正解
- 合計スコア800点以上

達成すると、画面いっぱいに紙吹雪が降り注ぎ、「天才の領域」の認定証が表示されます！
